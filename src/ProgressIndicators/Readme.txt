The ProgressIndicator sample demonstrates how to use the Connected Services core BusyIndicator to display general progress and a context sensitive progress indicator familiar to Windows users.

To use the styles referenced in the SpinnerControl.xaml, you’ll need a reference to Microsoft.VisualStudio.Shell.14.0, which ships as part of the Visual Studio SDK

Add a reference to your provider project, and click extensions to find the Visual Studio SDK extensions
